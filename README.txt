Amerimart Price Check - iPhone + iPad + Web PWA

SAFARI CAMERA FIX INCLUDED
- Uses BarcodeDetector where supported.
- Uses Quagga2 fallback for iPhone/iPad Safari.
- For first load, internet is needed so the scanner library can load.

UPLOAD TO GITHUB PAGES
1. Upload all files to your GitHub repo root.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main, Folder: / root.
5. Open the GitHub Pages link in Safari.

INSTALL ON IPHONE/IPAD
1. Open the GitHub Pages link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

IMPORTANT CAMERA SETTINGS
- Use HTTPS GitHub Pages link, not a local file.
- In iPhone Settings > Safari > Camera, choose Ask or Allow.
- If camera does not open, refresh Safari and allow camera permission.


NEW IN THIS VERSION
- Search by product name as well as UPC/barcode.
- If multiple items match, tap the correct product from the list.
- Example searches: Fre 9mg Mint, Monster, Marlboro, Coke.
